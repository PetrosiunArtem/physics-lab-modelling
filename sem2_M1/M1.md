# M1. Основная задача электростатики

Пусть $\varphi_1, \varphi_2$ - потенциалы электродов.
Как-нибудь разобъём их на n и m элементов соответственно. Пусть $q_1, ..., q_n$ - заряды элементов первого электрода, $q_{n+1}, ..., q_{n+m}$ - второго.

1. Рассмотрим потенциал $i$-го элемента в поле других. Считаем другие элементы точечными зарядами.

$$ \sum_{\substack{j=1 \\[2pt] i \ne j}}^{n+m} \frac {k q_i} {|\vec x_i - \vec x_j|} = 
\begin{cases} \varphi_1, i \le n \\ \varphi_2, i > n \end{cases}$$

2. Решаем получившуюся систему из $n+m$ уравнений с $n+m$ переменными (вроде бы не нужно использовать $\displaystyle \sum_{i=1}^n q_i = -\sum_{i=1}^m q_{n+i}$)

3. Находим напряжонности электрического поля и потенциалы в пространстве между электродами

$$ \vec E(\vec x) = \sum_{i=1}^n \frac {kq } {{|\vec x - \vec x_i|}^2} \cdot \frac {\vec x - \vec x_i} {|\vec x - \vec x_i|} $$

$$ \varphi(\vec x) = \sum_{i=1}^n \frac {kq } {|\vec x - \vec x_i|} $$
